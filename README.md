
# 💰 UniPocket — Student Finance Tracker

A student expense tracking system with two implementations:
- **Flask + MySQL** (DBMS mini-project)
- **C++** (OOPs mini-project)

## Features
- Add/view income and expenses
- Category-wise tracking
- Student-friendly UI

## Tech Stack
Python | Flask | PyMySQL | MySQL | C++

## Run Locally
```bash
pip install flask pymysql
python app.py
```
